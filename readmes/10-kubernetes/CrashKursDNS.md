## Crashkurs: DNS

* DNS?
  * Domain Name System
* Sorgt dafür, dass ein Domainname zu einer IP-Adresse aufgelöst wird
* Wenn man eine Webseite aufruft (z.B. google.de):
  * Dann wird der DNS-Server abgefragt, zu welcher IP sich Browser verbinden soll
  * Und der Browser stellt dann an diese IP die entsprechende Anfrage
